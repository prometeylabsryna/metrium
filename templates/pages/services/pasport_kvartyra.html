{% t 'Що таке технічний паспорт на квартиру' 'Что такое технический паспорт на квартиру' %}

{% if current_language == 'ru' %} Технический паспорт на квартиру — официальный документ установленного образца, содержащий подробное описание объекта недвижимости: площадь, этажность, планировку, технические характеристики и состояние конструктивных элементов. Документ оформляется сертифицированным специалистом и регистрируется в Единой государственной электронной системе в сфере строительства (ЕДЕССБ). {% else %} Технічний паспорт на квартиру — офіційний документ встановленого зразка, що містить детальний опис об'єкта нерухомості: площу, поверховість, планування, технічні характеристики та стан конструктивних елементів. Документ оформлюється сертифікованим експертом та реєструється у Єдиній державній електронній системі у сфері будівництва (ЄДЕССБ). {% endif %}

{% if current_language == 'ru' %} Без актуального техпаспорта невозможно провести ни одну юридическую операцию с квартирой: продажу, дарение, обмен, оформление наследства, приватизацию или ипотеку. {% else %} Без актуального техпаспорта неможливо провести жодну юридичну операцію з квартирою: продаж, дарування, обмін, вступ у спадщину, приватизацію або оформлення іпотеки. {% endif %}
